Box Model:

1. Block characteristic: Content, Padding, Border, Margin
2. Inline elements characteristic: no new start, contains data or other inline level elements
3. Margin: you can find it on the left, right, top and bottom border | Bsp: morgin-top: 100px;
4. Padding: Difference: space inside the border, margin space outside the border | Bsp: padding-top: 100px;

Classes and IDS:

1. (p).class { background-color: yellow; } | 2nd method: >p class="center large">This paragraph refers to two classes.>/p<
2. (p)#id { background-color: yellow; }
3. Start tag class: >p class<
4. Start tag id: >p id<
5. #header.callout { } | ID
6. #header .callout { } | Class
7. Classes can select more elements
8. IDs can only take one element
9. Classes multiple times in one page, Ids only once
